## ユーザ

操作する人間。
(例)ユーザがプレイするキャラクター

| 誤用         | 補足           |
| ------------ | -------------- |
| プレイヤー   |                |
| カバディスト |                |
| アクター     |                |
| ユーザー     | 長音は省略する |

## キャラ

3Dモデリングされた操作される選手モデル。
(例)キャラ選択

| 誤用         | 補足 |
| ------------ | ---- |
| キャラクタ   |      |
| キャラクター |      |
| アクター     |      |
| ユニット     |      |
